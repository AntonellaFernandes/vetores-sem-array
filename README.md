# Estruturas de Dados

Este projeto reúne a resolução de uma atividade prática desenvolvida em JavaScript utilizando conceitos essenciais de estruturação de dados: **Vetores**. Essa estrutura é uma das bases fundamentais da linguagem, permitindo agrupar, organizar e manipular informações de forma lógica e eficiente.

A aplicação dessas estruturas confere organização aos algoritmos, simulando como dados reais são gerenciados na memória antes de serem processados ou exibidos.

---

### Atividades Desenvolvidas

> **Orientação Geral:** Desenvolva os seguintes programas em javaScript sem utilizar de funções para arrays, e realize o upload dos arquivos para seu repositório remoto utilizando git:

* **#1 - Média de idade**
Crie um programa que leia a idade de 8 pessoas e guarde-as em um vetor. No final, mostre: Qual é a média de idade das pessoas cadastradas. 

* **#2 - Filtro de maiores de 25 anos**
Crie um programa que leia a idade de 7 pessoas e guarde-as em um vetor. No final, mostre: Em quais posições temos pessoas com mais de 25 anos. 

* **#3 - Filtro de múltiplos de 10**
Escreva um programa que leia 15 números e guarde-os em um vetor. No final, mostre o vetor inteiro na tela e em seguida mostre em que posições foram digitados valores que são múltiplos de 10. 

---

### Considerações Finais

Esta atividade é um passo essencial para compreender como o JavaScript estrutura informações no mundo real. Dominar a manipulação de vetores nos prepara para lidar com fluxos de dados mais complexos e dinâmicos em projetos futuros.
